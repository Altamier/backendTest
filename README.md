# backendTest
#Servidor simples usando node, expresse e mongo.
  
## Tecnologias

- Express
- REST API
- MongoDB
 

## Instalação

- Clonar o projeto
- Alterar a DATABASE_URL no .env
- npm install
- npm start

### Rotas

-  http://localhost:3000
  - /motoristas/testar
  - /motoristas/create
  - /motoristas/id/:id
  - /motoristas/:id
  - /motoristas/todos
  - /motoristas/veiculos/:id
  - /veiculos

#### Postman

- Instalar o Postman
- Importar a collection que está em _tests_
